# Complete List of AI Architectures

A progressively growing catalog of every AI architecture, filterable by tag, name, and year.

## How to run

Open `index.html` in a browser. No build step or server required.

## Tech

- HTML, CSS, JS (separated, no framework)
- Data stored in `js/data.js` as a plain array

## Structure

```
index.html
css/style.css
js/data.js    — architecture dataset
js/app.js     — filtering & rendering logic
docs/         — project documentation
```
